# Ice Shooter Garden Game

请帮我开发一个 2D 横版网页小游戏。

## 游戏目标
玩家控制一个寒冰射手，在花园里跳跃、发射冰弹，击败 5 个不同体型的僵尸。全部击败后进入胜利庆祝画面。

## 场景
- 卡通花园
- 草坪
- 栅栏
- 房子边缘
- 阳光明亮
- 适合小朋友玩的轻松风格

## 玩家角色
两个可选寒冰射手：
1. 寒冰射手 A：蓝色、小巧、可爱
2. 寒冰射手 B：蓝色、冰晶更多、攻击感更强

能力：
- 左右移动
- 跳跃
- 发射冰弹
- 冰弹击中敌人后造成伤害并减速

## 敌人
5 个僵尸，外形高矮胖瘦不同：
1. 普通僵尸
2. 瘦高僵尸
3. 胖僵尸
4. 矮小僵尸
5. 戴桶僵尸

## 操作
- A / D：左右移动
- W / Space：跳跃
- J：发射冰弹

## UI
- 开始游戏
- 选择角色
- 生命值
- 当前波次
- 暂停按钮
- 胜利结算页

## 胜利结局
打败所有僵尸后，显示庆祝画面：
寒冰射手和朋友们在花园里吃冰激凌庆祝。

## 技术要求
- 使用 React + HTML Canvas
- 可以直接运行
- 文件结构清晰
- 代码适合新手继续修改
- 先使用简单几何图形或占位图实现玩法，后续我会替换 PNG 素材
