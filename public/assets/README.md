# Game Art Assets

Place optional PNG game artwork at these public paths:

- `/assets/background/garden_bg.png`
- `/assets/plants/ice_shooter.png`
- `/assets/plants/ice_shooter_2.png`
- `/assets/zombies/zombie_normal.png`
- `/assets/zombies/zombie_cone.png`
- `/assets/zombies/zombie_bucket.png`
- `/assets/zombies/zombie_runner.png`
- `/assets/zombies/zombie_big.png`
- `/assets/bullets/ice_bullet.png`
- `/assets/ui/victory_page.png`

If a PNG is not present yet, the game keeps using its built-in placeholder drawing.
